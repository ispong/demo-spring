### Installation Prerequisites

- Java 1.8.+
- Gradle 6.5.+
- Git 2.22.+

### Build and Package

```
git clone -b latest https://github.com/ispong/spring-oxygen
cd spring-oxygen
gradle build
gradle publishToMavenLocal
```
