### Config Freecode

```
oxygen:
  freecode:
    author: ispong                                       # 开发者姓名
    module-path: com.isxcode.demo.oxygen                 # 文件生成地址

spring: 
  freemarker:
    enabled: true                                        # 开启freemarker

server:
  tomcat:
    uri-encoding: UTF-8                                  # 如果报错 Invalid character found in method name
```
